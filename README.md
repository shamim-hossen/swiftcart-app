# swiftcart-app

## 1) What is the difference between null and undefined?
 undefined মানে ভ্যালু দেওয়া হয়নি। 
 null মানে ইচ্ছে করে ভ্যালু ফাঁকা রাখা হয়েছে।
## 2) What is the use of the map() function in JavaScript? How is it different from forEach()?
 map() ডেটা নিয়ে সম্পূর্ণ নতুন অ্যারে তৈরি করে। 
 forEach() শুধু লুপ করে, নতুন কোনো অ্যারে বানায় না।
## 3) What is the difference between == and ===?
 == শুধু ভ্যালু সমান কি না চেক করে।
 === ভ্যালুর সাথে ডেটা টাইপও চেক করে।
## 4) What is the significance of async/await in fetching API data?
 API থেকে ডেটা আসা পর্যন্ত কোডকে আটকে রেখে কাজ করতে async/await ব্যবহার করা হয়।
## 5) Explain the concept of Scope in JavaScript (Global, Function, Block)?
 Scope হলো ভেরিয়েবল কোথায় ব্যবহার করা যাবে। 
 Global সব জায়গায় ব্যবহার করা যাবে। 
 Function শুধু ফাংশনের ভেতর ব্যবহার করা যাবে।
 Block শুধু { } এর ভেতর ব্যবহার করা যাবে।
